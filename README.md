Add a text to any .png or .jpeg picture using this command for the console written in C#.
Just download the file addText.exe into any folder and execute the file.
addText.exe admits two parameters: the .png file and the text you want to add to the picture.
Example:
addtext.exe background.png "javier928"
Result:
The text "javier928" will be placed in the picture..
Find the output.png file in your folder.
